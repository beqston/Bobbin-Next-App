Bobbin is a modern web application built with Next.js and styled with shadcn/ui — delivering a fast, accessible, and beautifully crafted user experience.
Built on the latest Next.js App Router with React Server Components, Bobbin leverages shadcn/ui's composable component library to maintain a consistent, polished design system. Whether you're shipping features fast or fine-tuning pixel-perfect interfaces, Bobbin's stack keeps things clean and scalable.
Tech Stack

Framework: Next.js (App Router)
UI Components: shadcn/ui
Styling: Tailwind CSS
Language: TypeScript

## Getting Started

First, run the development server:

```bash
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
